# Rundong Li · Academic Homepage

Personal academic website for [Rundong Li](https://orcid.org/0009-0009-8288-3049), published with GitHub Pages.

## Update the site

- Edit profile text, research themes, publications, and education in `index.html`.
- Edit visual styling in `styles.css`.
- The site uses no build step or external JavaScript dependencies.

## Preview locally

```powershell
python -m http.server 8000
```

Then open `http://localhost:8000`.
